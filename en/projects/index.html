Projects

Demarches-villages.fr : an online formalities for small municipalities

I couldn’t find a simple online formalities solution dedicated to the small municipalities so I created a service : demarches-villages.fr. During the conception, the emphasis was on the simplicity. One of the main goal was to limit as much as possible the inscription duration for the cities. By doing so, I hopefully encourage a maximum of cities that currently doesn’t use an online formalities solution due to their complexity to offer this service to their constituents. Like all of my projects, its use is free and the source code is publicly available.

Symphonium : A tool to help learning the piano

Inspired by Synthesia (proprietary software) and PianoFromAbove (open source but deprecated), it is the only open source and cross platform software that helps you learn the piano by vizualizing MIDI files and comparing thoses files to the notes you play on a MIDI keyboard. I created it as an alternative to Synthesia because every tool designed to help learning any skill should be free! For more information, you can check the project website.

BOARR : A benchmark for obstacle avoidance for drones

Created during my thesis, it is the first of its kind in aerial robotics. Its main goal is to main purpose is to test and compare the multiple existing avoidance algorithm for quadrotors with the hope that comparing the results could help choosing promising directions for future developments on autonomous obstacle avoidance. For more information, I redirect yourself to the dedicated Github repository.

[Ongoing] : InfoAsso, listings and tutorials of free softwares usefull to manage an association and InfoVillage, listings and tutorials of free softwares usefull to manage a small city

I would like to allow every person in charge of an association or a small collectivity to find and install all the free numerical tools he needs quickly and efficiently. Those websites would help reaching this goal by avoiding a lenghty and arduous research and by providing installation tutorials.

Currently, for the associations, if someone has a bit of IT knowledge and a good internet connexion, an internet cube seems to be a great solution which open the possibility to host services for every needs. Without any technical knowledge, there are some structures, such as Indie Host that offer a turnkey solution.

There seems to be a lot less choice for small collecvities. I’m currently focusing on their needs and on identifying where there is a lack of solution. This led to the development of demarches-villages.fr but there is still a lot to do. The addulact, a representative association, is a good place to look for more informations and OpenMairie is offering some solutions for very specific needs.
